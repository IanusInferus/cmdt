盟军敢死队3D .Mbi文件浏览器(3D .Mbi Viewer) ver.1.5.5

- by <NeoRAGEx2002>  2007.9.29


1. 运行时软、硬件环境

a. Windows XP pro SP2(我只测试了xp下的运行情况，其他的没有测过)
b. Microsoft .Net framework 2.0(22.4MB)，下载地址为:
   http://download.microsoft.com/download/5/6/7/567758a3-759e-473e-bf8f-52154438565a/dotnetfx.exe
c. DirectX October '06版(55.6MB)，下载地址为:
   http://download.microsoft.com/download/d/4/6/d46cc24d-33df-4727-aa89-9512513c67d3/directx_oct2006_redist.exe
d. 支持DirectX 8.0的显示加速卡，32M显存，1024*768*32位真彩色、24位Z深度、双线性及各向异性材质过滤、2x全屏抗锯齿FSAA


2. 键盘热键

J、K: Camera距离远近
C   : 切换显示坐标系原点位置
F   : 线架、贴图切换
X   : 坐标轴显示开关
A   : 切换全屏抗锯齿功能
E   : 导出OBJ/MTL文件及所有贴图
R   : 恢复默认视角
O   : 查看其他.MBI文件
Q   : 退出


3. 鼠标控制

左键拖放 : 自由旋转
右键     : 线架、贴图切换
中键     : 切换全屏抗锯齿
滚轮     : Camera距离远近
